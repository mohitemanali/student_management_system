# Student Management System

## Project Description
The Student Management System is a simple Python project that allows users to manage student records.  
Users can add student details, view all students, and delete student records using a command-line interface.

This project helps beginners understand Python concepts like dictionaries, functions, loops, and conditional statements.

---

## Features
- Add new student details
- View all student records
- Delete student records
- Simple command-line interface

---

## Technologies Used
- Python 3
- Command Line Interface (CLI)

---

## How to Run the Project

Step 1: Clone the repository
git clone https://github.com/your-username/student-management-system.git

Step 2: Open the project folder
cd student-management-system

Step 3: Run the program
python student_management.py
